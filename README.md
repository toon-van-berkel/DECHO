# Echo-Shards

> A fantasy game about privacy, where you help citizens reclaim the memories a council keeps "safe."

`CLE4` · `Team 9` · `Games`

## About this project

This is a school project for the course **CLE4 - Games** at Hogeschool Rotterdam (CMGT). The goal of the course is to design and build a real, playable game while applying object-oriented programming (OOP).

Echo-Shards is a 2D top-down singleplayer Excalibur game about privacy in a world where the government holds people's thoughts and memories. A body called the Echo Council collects and stores every citizen's Echo Shards in the name of safety, and not everyone accepts that. As the player, you move through this world and help citizens recover or repair the shards that were taken from them, working through conversations and interactions with the NPCs you meet. The fantasy setting turns an abstract question about surveillance and consent into something you can see, hold, and act on.

- **Social theme:** Privacy (social impact)
- **Player type:** Socializer (interaction-driven, helping citizens through NPC dialogue)
- **Fun factor:** Fantasy
- **Engine:** [ExcaliburJS](https://excaliburjs.com/)
- **Tooling:** [Vite](https://vitejs.dev/)
